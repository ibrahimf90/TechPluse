# TechPluse
TechPulse is a sleek, responsive website for a premium tech and lifestyle store, featuring modern design elements and intuitive navigation for showcasing cutting-edge products.

##Features
Responsive navigation menu with mobile toggle
Hero section with call-to-action button
Shopping cart indicator in navigation
Modern, minimalist design

##Technologies Used
HTML5
CSS3
Font Awesome (for icons)
